# NovaBytes Batch Renamer

## User Guide

---

# Getting Started

Launch the application.

You can load files using one of three methods:

- Add Files
- Add Folder
- Drag & Drop

---

# Renaming Files

## Prefix

Adds text before every filename.

Example

```
IMG001.jpg

↓

Holiday_IMG001.jpg
```

---

## Suffix

Adds text after the filename.

```
IMG001.jpg

↓

IMG001_Edited.jpg
```

---

## Numbering

Enable numbering to generate sequential filenames.

Options

- Start Number
- Increment
- Padding

Example

```
Start = 1

Padding = 3

↓

001
002
003
```

---

# Rename Files

Click **Rename Files** to apply all changes.

---

# Undo

Undo reverts the last rename operation.

Shortcut

```
Ctrl + Z
```

---

# Redo

Redo reapplies the last rename.

Shortcut

```
Ctrl + Y
```

---

# Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Ctrl + O | Add Files |
| Ctrl + Shift + O | Add Folder |
| Ctrl + R | Rename Files |
| Ctrl + L | Clear |
| Ctrl + Z | Undo |
| Ctrl + Y | Redo |
| Ctrl + A | Select All |
